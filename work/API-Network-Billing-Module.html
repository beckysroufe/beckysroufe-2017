API Network Billing Module

Web ApplicationUX Architecture, UI Design & Front End Development

To support the API Network Endpoint Manager, we created a billing module and monetization component to our platform where developers could manage APIs, API usage and pay providers for service. I helped guide our product design from the initial research + planning stages through design and into development.

1Business Objectives

Through initial research on the API Network Endpoint Manager, we identified a need for a key feature within our application that would allow developers to pay fees to providers for API data access and usage that would also compensate providers for their services. I assisted the business team in building the requirements based on our user research and aligning it with our business goals and objectives.

2User Flows

I put together a user flow to identify the various needed steps the end user would need to accomplish various key flows and features within this application. This also helped us continually focus on streamlining flows and work towards the goal of keeping user tasks as minimal and simple as possible, thus reducing effort and time needed to accomplish task.

3User Interface Designs

I built high-fidelity prototypes in Adobe Photoshop to help bring our wireframes to life through visual design and used InVision to help simulate the end user experience so we could better test and validate the early concept before moving into development.

3Front End Development

I collaborated with another developer on our team to help build our product, with my role primarily focusing on translating our visual design mockups into a UI prototype using HTML, CSS and JavaScript within a Backbone.js environment utilizing Marionette.
